# connecter1
Connecter1 is a memory game where each emoji hides a different fruit with its matching pair. Your goal is to find all the pairs in the shortest time possible, and if you win, you’ll unlock a really cute GIF!

I created this game inspired by @giridhar7632’s workshop, but I added my own special touches, such as an instructions container that appears when you open the game, along with sound effects for both mistakes and correct matches.

I built it using HTML, CSS, and JavaScript to give it an interactive feel.

This is a really fun game to play whenever you have some spare time while also exercising your memory. I really hope you enjoy it!
